
# AI Resume Screening Assistant 

An efficient resume screening web tool designed for HR and recruitment teams. Simply input the job description (JD) and bulk-upload PDF resumes. It parses text content, calculates matching scores based on weighted algorithms, and generates candidate rankings with in-depth analysis. Supports both English and Chinese.

# AI 简历筛选助手

一个用于HR/招聘团队的简历筛选网页：输入JD，批量上传PDF简历，解析文本并按权重计算匹配度，生成候选人排行榜与详细分析；支持中英文。

test dataset: https://tianchi.aliyun.com/dataset/201566
